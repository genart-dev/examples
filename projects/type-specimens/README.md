# Type Specimens

Typography specimen sheets showcasing `@genart-dev/plugin-typography`.

![Type Specimens](renders/specimen-sheet.png)

## Scenes

| # | Scene | Description |
|---|-------|-------------|
| 1 | Font Stack Catalog | Each built-in font at multiple sizes |
| 2 | Pangrams | Classic pangrams in different weights and styles |
| 3 | Scale & Hierarchy | Typographic scale — display through caption |
| 4 | Text Blocks | Paragraph layout with left/center/right/justify alignment |
| 5 | Character Grid | Full character set grid |
| 6 | Specimen Sheet | Combined overview |

## Plugins

- `@genart-dev/plugin-typography` — `textLayerType`, `BUILT_IN_FONTS`, `resolveFontStack`

## Usage

```bash
npm install
node render.cjs
```

Output goes to `renders/`.
