# Figure Studies

Life drawing studies showcasing `@genart-dev/plugin-figure` and `@genart-dev/plugin-poses`.

![Figure Studies](renders/figure-sheet.png)

## Scenes

| # | Scene | Description |
|---|-------|-------------|
| 1 | Mannequin Styles | Same standing pose in 5 methods — stick, bean, blocks, loomis, anatomy |
| 2 | Gesture Drawing | Line of action + rhythm curves — smooth, angular, organic, taper variations |
| 3 | Proportion Systems | All 7 systems side-by-side — child (4-6h), average (7.5h), ideal (8h), heroic (8.5h), fashion (9h) |
| 4 | Head Construction | Loomis head from 7 angles + Asaro and Simple methods |
| 5 | Posed Skeletons | 3 poses (standing, sitting, dynamic) with stick/proportional/silhouette styles and color modes |
| 6 | Figure Sheet | Combined overview of all scenes |

## Plugins

- `@genart-dev/plugin-figure` — `mannequinLayerType`, `gestureLayerType`, `headLayerType`, `proportionGridLayerType`
- `@genart-dev/plugin-poses` — `skeletonLayerType`

## Usage

```bash
npm install
node render.cjs
```

Output goes to `renders/`.
